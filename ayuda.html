Ayuda y Guía Completa para la Web de Grado


Cómo usar la página Inicio

La página principal te da la bienvenida y te muestra imágenes destacadas y noticias recientes sobre el concejo de Grado.

Qué encontrarás:

  • Un carrusel de fotos que muestra los paisajes, monumentos y lugares interesantes del concejo.
  • Una sección de noticias con los últimos eventos y novedades importantes.

Para interactuar: Puedes hacer clic en el botón Leer más de cada noticia para abrir la página original y obtener más detalles. Para pasar de imagen en el carrusel, pulsa sobre "Siguiente" o "Anterior".

Explora las imágenes para conocer mejor el concejo y descubrir sitios que podrías visitar.



Cómo usar la página Gastronomía

Esta sección está dedicada a la gastronomía típica del concejo de Grado, para conocer los sabores y tradiciones culinarias locales.

Qué verás:

  • Información sobre los platos tradicionales y sus ingredientes.
  • Un vídeo que muestra el concejo de Grado.
  • Un audio que puedes escuchar para conocer música tradicional de Grado.

Recomendación: Puedes usar esta página para inspirarte si planeas cocinar platos del concejo o simplemente para aprender sobre su cultura gastronómica.



Cómo usar la página Rutas

En esta página puedes explorar las diferentes rutas turísticas disponibles en el concejo de Grado.

Pasos para usarla:

  1. Selecciona el archivo XML que contiene la información de las rutas. Esto suele ser un archivo proporcionado por la web o descargado previamente.
  2. Una vez cargado el archivo, la página mostrará automáticamente los detalles de cada ruta, como una descripción completa, la valoración que otros usuarios han dado, y las personas capacitadas para realizarla.
  3. Verás un mapa interactivo de Google Maps donde se dibuja el recorrido exacto de la ruta para que puedas visualizar el camino que debes seguir.
  4. También se mostrará la altimetría de la ruta en un gráfico SVG, que indica las subidas y bajadas de altura, para que puedas prepararte físicamente para la dificultad del recorrido.

Consejos: Asegúrate de que el archivo XML esté en el formato correcto y contenga la información necesaria. Si el mapa o la altimetría no se cargan, revisa tu conexión a internet o intenta con otro archivo.



Cómo usar la página Juego

El juego es un cuestionario con 10 preguntas de opción múltiple relacionadas con el concejo de Grado y sus rutas.

Cómo jugar:

  1. Lee cada pregunta atentamente.
  2. Selecciona una respuesta entre las opciones disponibles haciendo clic en el círculo al lado de la opción.
  3. Después de responder cada pregunta, pulsa el botón Siguiente para avanzar a la siguiente pregunta.
  4. Cuando respondas la última pregunta, pulsa Finalizar para ver tu resultado.

Al finalizar: Se mostrará tu puntuación total y un resumen de todas las preguntas.

Para cada pregunta, podrás ver si tu respuesta fue correcta o incorrecta. Si te equivocaste, aparecerá la respuesta que marcaste y la respuesta correcta para que aprendas.

Consejo: Tómate tu tiempo para leer bien las preguntas y respuestas. No hay límite de tiempo, y puedes aprender con el resumen final.



Cómo usar la página Meteorología

Esta sección te informa sobre el estado del tiempo actual y la previsión en el concejo de Grado, para ayudarte a planificar tus actividades al aire libre.

¿Qué puedes ver?

  • Temperatura actual en grados Celsius.
  • Velocidad del viento.
  • Pronóstico del tiempo para los próximos siete días, con detalles como temperatura máxima y mínima.

Cómo interpretar la información: Usa esta página para saber si es buen momento para salir a caminar o visitar alguna ruta. Si ves pronóstico de lluvia o viento fuerte, es mejor planificar otro día.
